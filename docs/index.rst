Welcome to MPI_CBUS documentation!
===================================

**MPI_CBUS** is a Python library that implements the MERG CBUS Protocol as
specified in `https://www.merg.org.uk/merg_wiki/lib/exe/fetch.php?media=cbus:developer_6b.pdf`.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the software.

.. note::

   This project is under active development.
